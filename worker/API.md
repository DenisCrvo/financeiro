# API — financeiro-api (Cloudflare Workers)

> A documentação completa da API foi movida para
> **[`docs/API_REFERENCE.md`](../docs/API_REFERENCE.md)**, na raiz do
> projeto, para evitar duas versões divergentes do mesmo conteúdo.

Este arquivo permanece como um ponteiro porque `worker/API.md` é o local
"óbvio" onde alguém navegando só a pasta `worker/` procuraria a documentação
da API — mas o conteúdo (endpoints, payloads, exemplos, códigos de erro) vive
exclusivamente em `docs/API_REFERENCE.md`. Ao alterar qualquer rota,
atualize **apenas** lá.

Veja também:
- [`docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md) — roteamento e fluxo de requisições no Worker
- [`docs/BUSINESS_RULES.md`](../docs/BUSINESS_RULES.md) — regras de negócio por trás de cada rota
- [`docs/DATABASE.md`](../docs/DATABASE.md) — schema consultado por cada rota
